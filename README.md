## Stack

- **React 19** + **TypeScript**
- **Material UI v7** for components and theming
- **React Router v7** for client-side routing
- **Supabase** for database and auth
- **Netlify** for hosting and serverless functions
- **Cloudinary** for image storage and optimization

## Features

- Browse and search the full recipe collection
- Filter by ingredient, region, type, source, and rating
- Recipe detail pages with ingredients, instructions, and a print view
- QR code generation on each recipe for easy sharing
- Shopping list (authenticated users)
- Admin tools for adding and editing recipes
- Installable as a PWA
